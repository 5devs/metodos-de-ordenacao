Selection Sort

Seleciona um elemento e o compara com os demais, partindo da esquerda para a direita, trocando as posições quando necessário.

                                                                  Bubble Sort

                                                                  Compara os itens em pares, da esquerda para a direita, trocando-os quando necessário. Repete o movimento até que a coleção esteja corretamente ordenada.

                                                                    Quicksort

                                                                    Utilizando um elemento como guia (chamado de pivô), os itens restantes são separados em dois grupos, os menores e os maiores que o pivô; os valores à esquerda e à direita, respectivamente.

                                                                    Desses dois vetores criados, a lógica é aplicada repetidamente, até que os itens encontrem-se ordenados.

                                                                      Gnome Sort

                                                                      Baseado no Bubble Sort, também compara os elementos vizinhos da esquerda para a direita. Quando um item deve ser trocado, o algoritmo compara da direita para a esquerda para encontrar a posição correta do elemento sendo reposicionado.

                                                                      Esse método evita que o algoritmo percorra elementos já ordenados.

                                                                                                                        Comb Sort

                                                                                                                        Também baseado no Bubble Sort, o principal intuito desse algoritmo é eliminar os elementos tartaruga mais rapidamente, utilizando um gap maior para cada iteração.

                                                                                                                        A ideia de utilizar um gap maior que 1 é parecida com a do Shell Sort, mas o mesmo é uma modificação do Insertion Sort, não do Bubble Sort.

                                                                                                                                                                              Cocktail Sort

                                                                                                                                                                              Também visando diminuir os problemas dos elementos tartaruga no Bubble Sort, o Cocktail Sort (também conhecido como Bubble Sort Bidirecional) compara os elementos em pares, da esquerda para a direita, e ao chegar no fim volta comparando os elementos da direita para a esquerda, invertendo a comparação.
